My Flix

the web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies


Business Logic URL HTTP Method Request body data format Response body data format
Return list of all movies to the user /movies GET None A JSON object holding data about all the movies
Return data about single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie
Return data about a genre by name/title /movies/genres/:genre name GET None A JSON object holding data about the movies in the genre
Return data about a director by name /movies/directors/:director name GET none A JSON object holding data about the movies by the directon
Allow new users to register /users POST A text message indicating that the user has successfully registered
Allow users to update their user info /users/:name/:password/:email/:birthdate PUT A JSON object holding data about the user to add A JSON object holding data about the user that was added
Allow users to add a movie to their list of favourites /users/:name/movies/:title PUT A text message indicating the movie and details that was added
Allow users to remove a movie from their list of favourites /users/:name/movies/:title DELETE A text message indicating the movie was successfully deleted
Allow users to derigister /users/:userName DELETE A text message indicating that the user was succesfully removed
